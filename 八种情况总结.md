### 3. 投衣动画：把情绪词变成衣服

| 焦虑                        | 焦虑衣服     |
| 愤怒                        | 愤怒袜子     |
| 疲惫                        | 疲惫外套     |
| 委屈 / 难过 / 想哭          | 委屈毛衣     |

| 内耗 / 纠结 / 自责 / 想太多 | 叠加线团效果 |

| 优先级 | 情绪      | 显示规则                     |
| ------ | --------- | ---------------------------- |
