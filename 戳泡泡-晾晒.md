### 5.微距戳泡泡

#### 页面元素与空间布局

* **背景特写** ：模糊的脏衣服背景，整个屏幕充满淡淡的、正在旋转的蓝色水流微波特效。
* **画面前景** ：3个大泡泡静止在空中。每个泡泡的中心，都用带有锯齿感、尖锐的黑色粗字体，写着一句极端的坏话（如：“我必须立刻把所有事做好！”、“我已经完全来不及了！”）。

对应如下：

| 情绪      | 极端想法示例                                                                      |
| --------- | --------------------------------------------------------------------------------- |
| 焦虑      | **我必须立刻把所有事都做好！**/**只要有一点没完成，我就完蛋了！**     |
| 愤怒      | **凭什么所有人都这样对我！**/**我现在真的一句话都不想忍了！**         |
| 疲惫      | **我真的一点力气都没有了！**/**为什么我永远都休息不过来！**           |
| 迷茫      | **我完全不知道自己在干什么！**/**别人都在往前走，只有我卡在原地！**   |
| 委屈      | **明明我已经很努力了，为什么没人看见！**/**为什么受伤的总是我！**     |
| 内耗      | **我是不是又做错了什么！**/**我脑子里一直停不下来，快被自己烦死了！** |
| 孤独      | **好像根本没有人真的懂我！**/**我就算消失了，也不会有人发现吧！**     |
| 崩溃/发疯 | **我现在真的要炸了！**/**我控制不住了，什么都不想管了！**             |

#### 元素出现与动态表现

* **自发漂浮呼吸** ：3个大泡泡在各自的坐标点上，保持着极微弱的上下浮动和“放大/缩小”的呼吸感（缩放比例在 0.98 到 1.02 之间），模拟泡泡在水流中的轻盈感。

#### 触发交互与原位翻转转化

* **点击触发** ：玩家伸手去点击其中任意一个大泡泡。
* **破裂动效** ：被点击的泡泡瞬间播放一个 3 帧的“圆形碎裂四散线条”视觉效果，并伴随清脆的“啵”的一声。
* **【核心变化】原位转化非消失** ：泡泡的圆形外壳虽然炸裂消失了，但是 **里面的尖锐黑色文字完全没有消失，也没有位移** 。
* 文字的原位坐标上，瞬间触发一个“180度垂直翻转（Flip）”或“原位渐隐换字”**的视觉特效。原本尖锐、刺眼的黑色极端坏话，在原位瞬间转变成了一行**字体圆润、散发着金色柔光、带有温暖方块卡片底衬的“温和治愈词卡”（如：“我必须立刻把所有事做好” ➔ 原位变成 ➔  **“我可以先走好第一步”** ）。
* 玩家顺次将 3 个泡泡全部点击戳破，屏幕上最终留下 3 块散发着柔光的、静态的温馨词语卡片。

#### 视角流转（滚筒内舱 ➔ 全局还原）

* **触发瞬间** ：当第 3 个泡泡被戳破、最后一张卡片完全转化完成的瞬间，滚筒内的强光骤放。
* **镜头运镜（Zoom Out）** ： **镜头触发极速的向后拉远（缩放拉伸）** 。原本巨大的滚筒内舱快速缩小，洗衣机的外壳化为一层柔和的UI线框或直接淡出。
* **转场动效** ：整个暗色调的内舱空间如同迷雾消散，屏幕瞬间恢复为干净、清爽、明亮的 **全局底色** 。在此过程中，洗净的衣服与 3 张词卡脱离合并状态，顺畅地流转到各自的最终归宿位置。

### 6.焕然重组 ——【全局情绪梳理视角】

#### 页面元素与空间布局

* **画面上半部分【核心视觉区】** ：那件原本脏乱、代表负面情绪的衣服，此时已经 **彻底蜕变** 。它变得干净整洁、平整如新，并且带有 **治愈开心的微表情** 。衣服静止或微幅悬浮在屏幕正中上方。
* **画面下半部分【互动操作区】** ：一排整齐摆放的 5 张小长方形卡片。其中包括刚刚清洗转化出来的 3 句温馨词卡，以及系统自带的两张骨架词卡（固定为【我】和【但是我觉得】）。这 5 张卡片的横向顺序是  **杂乱且随机打乱的** ，等待用户进行下一步的“情绪拆解与重排”。

#### 元素出现与动态表现

* **衣服的动态** ：经历清洗后，这件干净开心的衣服在画面上方呈现出轻盈的“呼吸感”上下浮动，偶尔伴有轻微的左右晃动（类似衣物在微风中舒适摆动的动效），四周可自带淡淡的微光或星星动效，视觉上不断向用户传递“被治愈”的正向反馈。
* **底部卡片滑入** ：当全局场景定格、衣服就位后，下方的 5 张词语卡片像扑克牌开牌/发牌一样，带有清脆的弹簧音效，从屏幕最底端 **整齐地向上滑入并一字排开** 。
* **交互引导** ：卡片排列完毕后，卡片上方或边缘出现微弱的呼吸走马灯提示，明确暗示用户可以通过**拖拽**这些卡片，开始在上方进行句式的重新组合。
