| 项目             | 最终决定                                                                                                                                         |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| 作品名           | 《把今天洗一下》                                                                                                                                 |
| 一句话介绍       | 一个把坏情绪变成脏衣服、通过多种交互方式的洗衣流程生成情绪小票的治愈向互动小游戏                                                                 |
| 必做页面         | 首页（素一点）、洗衣房界面->输入情绪动画(聚焦脏衣篓)->投衣动画->清洗页(聚焦洗衣机)->小票页（情绪洗衣单）                                         |
| 可选页面         | 衣柜页 / 我的小票页                                                                                                                              |
| 情绪类型         | 焦虑、愤怒、疲惫、迷茫、委屈、内耗、孤独、崩溃/发疯                                                                                              |
| 洗涤模式         | 先写一种今日洗涤，后面有时间可以补充轻柔洗、速洗、疯狂洗等                                                                                       |
| 核心互动         | 肥皂（涂抹） → 滚筒（拖动旋转/摇晃手机） → 戳泡泡（点击） → 脱水（长按甩干） → 晾晒（拖动）                                               |
| 技术边界         | 当前版本重点实现单次情绪清洗流程，不涉及长期心理评估、医疗诊断或复杂社交系统。                                                                   |
| 最后 30 分钟原则 | 不加功能，只修 bug、错字、提交材料                                                                                                               |
| 网络技术         | 本项目可接入大语言模型 API，对用户输入的负面情绪文本进行情绪识别、攻击性表达提取、真实感受拆解与温和表达重组，从而提升结果生成的准确性和自然度。 |
