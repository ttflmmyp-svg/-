# 首页/欢迎页

### 页面目的

> 让用户 3 秒内知道：这是一个情绪洗衣房，可以开始洗今天的心事。

### 用户看到什么

标题：

> 欢迎来到情绪洗衣房

副标题：

> 今天有什么想法需要洗一洗？

NPC 气泡（准备两到三句轮换）：

> 脏一点也没关系，洗衣房就是干这个的。

主按钮：

> 开始洗衣

玩法暗示（洗衣机旁边）

输入一段心事，把它洗成更轻一点的样子

可选按钮（后续再补充）：

> 查看我的小票

### 用户能点什么

用户点击  **开始洗衣** 。

### 点击后发生什么

进入输入页。

### 异常情况

* 如果图片未加载，显示洗衣机 / NPC 占位图。
* 如果用户连续点击“开始洗衣”，只跳转一次。
* 如果小票页未完成，“查看我的小票”按钮暂不显示。

### 设计重点

首页重点不是解释规则，而是让用户觉得“这里安全、好玩、可以开始”。

信息要少，按钮要明显，NPC 要有陪伴感。

### 需要 B 实现什么

* 打开游戏默认显示首页
* 手机端居中显示
* 页面在手机端 390px 宽度下不横向溢出。
* 洗衣机显示
* 标题、副标题
* NPC气泡和洗衣机（NPC、洗衣机图片没给到时，先用占位块，不影响页面跳转）
* 主按钮明显，位于视觉中心下方。
* 点击“开始洗衣”后跳转到 `/input` 或 input 页面状态。
* 按钮点击需要有简单反馈，例如按下变暗、轻微缩放。

### 需要 C 提供什么

* 洗衣机主视觉图
* NPC 基础图
* 首页 NPC 台词 2-3 条
* - 脏一点也没关系，今天的心事我来洗。
  - 把烦人的想法丢进来，我们慢慢搓干净。
  - 不用整理得很漂亮，乱糟糟也可以开始洗。
* 首页背景风格建议
* 首页整体氛围关键词，例如：柔软、可爱、轻松、微脏但不压抑
