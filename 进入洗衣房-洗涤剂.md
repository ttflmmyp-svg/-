### 总体空间：情绪洗衣房

整个项目只有一个主场景：

> 一个洗衣房背景，里面有洗衣机、脏衣篓、NPC、小票机/柜台、衣柜/收纳柜等元素。

用户不是“跳页面”，而是在这个空间里完成完整流程。

### 1. 洗衣房初始状态

用户进入后看到完整洗衣房。

画面中心是洗衣机，整个画面氛围轻松一点。

文案：

> 欢迎来到情绪洗衣房
>
> 今天有什么心事需要洗一洗？

按钮：

> 开始洗衣

点击后，不是跳到一个完全新的输入页，而是 **镜头移动/聚焦到脏衣篓区域** 。

### 2. 输入情绪动画：聚焦脏衣篓

镜头慢慢推近到左下角的脏衣篓。

脏衣篓旁边出现输入框。

文案可以是：

> 把今天的情绪脏衣服丢进来吧。

输入框 placeholder：

> 比如：我今天很烦，感觉什么都做不好……

如果用户输入太短，比如“烦”“累”“不开心”，可以先不立刻进入清洗，而是让 NPC 轻轻追问一句：

> 1、怎么了，发生什么事了？
>
> 或者
>
> 2、这种感觉真的很累，觉得哪里都卡住了。
>
> 是有什么具体的事让你觉得做不好，还是最近整体状态不太对劲？

但这里不要太强制。可以给两个按钮：

> 再说一点
>
> 直接开洗

这样用户不会觉得被审问

### 3. 投衣动画：把情绪词变成衣服

用户确认后，输入内容变成一件皱巴巴的小衣服/布团/纸团。

| 输入识别情绪                | 最终显示衣服 |
| --------------------------- | ------------ |
| 焦虑                        | 焦虑衣服     |
| 愤怒                        | 愤怒袜子     |
| 疲惫                        | 疲惫外套     |
| 委屈 / 难过 / 想哭          | 委屈毛衣     |
| 迷茫 / 无助 / 不知道怎么办  | 迷茫围巾     |
| 孤独 / 被忽略 / 没人懂      | 孤独手套     |
| 崩溃 / 发疯 / 破防          | 崩溃睡衣     |
| 内耗 / 纠结 / 自责 / 想太多 | 叠加线团效果 |

#### 判断优先级

| 优先级 | 情绪      | 显示规则                     |
| ------ | --------- | ---------------------------- |
| 1      | 崩溃/发疯 | 只要出现强崩溃词，就优先显示 |
| 2      | 愤怒      | 出现明显愤怒、攻击、不爽表达 |
| 3      | 委屈      | 出现想哭、被误会、没人理解   |
| 4      | 焦虑      | 出现担心、紧张、压力、害怕   |
| 5      | 疲惫      | 出现累、困、没力气、熬夜     |
| 6      | 迷茫      | 出现不知道怎么办、没方向     |
| 7      | 孤独      | 出现一个人、没人陪、被忽略   |
| 8      | 内耗      | 不作为主衣服，作为叠加效果   |

输入框里的文字 → 缩成一件“脏衣服” → 掉进洗衣机

下一步转移视角自动移步到洗衣机

### 4.倒入洗涤剂

#### 洗涤剂设置

根据不同的情绪选择不同的洗涤剂，对应如下

| 情绪      | 洗涤剂名字                                      |
| --------- | ----------------------------------------------- |
| 焦虑      | **慢慢来洗衣液**/**先呼吸柔顺剂**   |
| 愤怒      | **消消气洗衣液**/**轻一点漂白剂**   |
| 疲惫      | **休息一下洗衣液**/**充充电柔顺剂** |
| 迷茫      | **走一步洗衣液**/**会清楚柔顺剂**   |
| 委屈      | **抱抱你洗衣液**/**不怪你柔顺剂**   |
| 内耗      | **别拧巴洗衣液**/**放过自己柔顺剂** |
| 孤独      | **有人在洗衣液**/**陪陪你柔顺剂**   |
| 崩溃/发疯 | **先暂停洗衣液**/**没关系柔顺剂**   |

* 当瓶子被拖拽到洗衣机上方，触发一个“瓶身倾斜 45 度”的动画。
* 同时，洗衣机内部 **直接播放一个预设好的“泡泡逐渐冒出”的 2D 帧动画/缩放动画** 。
* 衣服在洗衣机里做一个**上下轻微浮动、左右摇摆**的 Tween 动画（代码控制位移即可，完全不需要物理引擎），表现它在被洗涤。

#### 元素出现与动态表现

* **重力掉落** ：当视角定格时，那件脏衣服像从天而降一样，带有“物理重力感和微微弹跳”从屏幕最上方垂直掉落到画面正中央。
* **平滑滑入** ：对应的特效洗涤剂瓶子从置物架的右侧边缘，平滑地向左位移滑入到衣服旁边，并散发着一圈淡淡的金色外轮廓呼吸光芒，暗示玩家“点我”。

#### 触发交互与倒料动效

* **点击/拖拽触发** ：玩家用手指按住洗涤剂瓶子往衣服上拖拽（或直接点击瓶子）。
* **倒料动画** ：洗涤剂瓶子自动飘移到脏衣服的领口正上方， **瓶身自动倾斜 45 度** ，瓶口处出现几滴半透明的液体粒子向下滴落（持续1秒）。随后瓶子渐隐淡出消失。
* **预洗动效** ：脏衣服接触到洗涤剂后，原位播放 1.5 秒的“高频左右轻微晃动”动画（模拟被揉搓），并伴随着咕噜咕噜的水泡声。
* **泡泡自动浮现** ：晃动结束，衣服原位静止。从脏衣服的衣服口袋、领口、下摆三个位置， **自己缓缓飘出 3 个大大的、半透明的圆形大泡泡** 。这 3 个泡泡在屏幕中呈三角形散开，分别停留在屏幕的左上、中右、左下三个固定坐标点。

#### 视角流转（中景舱口 ➔ 微距泡泡）

洗涤剂瓶子彻底消失。 **镜头往内进行微距推近（Zoom In）** ，洗衣机舱门和脏衣服变成略微模糊的宏大背景， **视角100%聚焦在漂浮在空中的这 3 个大泡泡和里面的文字上** 。
